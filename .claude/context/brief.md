# Climate Indices — Brief

A Python library for computing standardized climate drought indices (SPI, SPEI, PET, Palmer, PNP).

## Key Features
- NumPy-based core with Numba acceleration
- xarray-native API with optional Dask parallelism
- CLI for batch NetCDF processing
- Supports monthly and daily time series

## Tech Stack
- Python 3.10–3.13
- NumPy, SciPy, Numba (core)
- xarray, Dask (scalable processing)
- pytest, ruff, mypy (quality)
